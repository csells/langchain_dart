# Multi-Provider Conversations

Share conversations seamlessly between different AI providers to leverage each one's strengths.

## Why Multiple Providers?

### Cost Optimization
```dart
// Simple tasks with cheap provider
final gemini = ChatAgent('google');
var result = await gemini.run('Summarize: $text', history: history);
history.addAll(result.messages);

// Complex tasks with powerful provider
final claude = ChatAgent('anthropic');
result = await claude.run('Analyze implications...', history: history);
history.addAll(result.messages);
```

### Capability Mixing
- **Claude**: Deep reasoning
- **GPT-4**: General capability + tools
- **Gemini**: Fast and cheap
- **Mistral**: European languages

### Tool Availability
```dart
// Start with Claude
final claude = ChatAgent('anthropic');
var result = await claude.run('I need weather data', history: history);
history.addAll(result.messages);

// Switch to OpenAI with tools
final openai = ChatAgent('openai', tools: [weatherTool]);
result = await openai.run('Get Seattle weather', history: history);
history.addAll(result.messages);

// Back to Claude for analysis
result = await claude.run('What should I wear?', history: history);
```

## Complete Example

```dart
Future<void> processCustomerRequest() async {
  final history = <ChatMessage>[];
  
  // 1. Quick triage with Gemini
  final triage = ChatAgent('google:gemini-2.0-flash');
  var result = await triage.run(
    'Customer says: "My order #12345 hasn\'t arrived"',
    history: history,
  );
  history.addAll(result.messages);
  
  // 2. Lookup with tools (OpenAI)
  final lookup = ChatAgent('openai', tools: [orderLookupTool]);
  result = await lookup.run(
    'Look up order status',
    history: history,
  );
  history.addAll(result.messages);
  
  // 3. Craft response with Claude
  final responder = ChatAgent('anthropic:claude-3-5-haiku');
  result = await responder.run(
    'Write a helpful customer service response',
    history: history,
  );
  history.addAll(result.messages);
  
  print('Response: ${result.output}');
}
```

## Best Practices

1. **Start cheap**: Use fast providers for simple tasks
2. **Escalate wisely**: Switch to powerful providers when needed
3. **Use tools strategically**: Pick providers with required tools
4. **Maintain context**: Always pass the full history
