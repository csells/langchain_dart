# Provider Selection

Dartantic offers flexible ways to select and work with AI providers, from static typed providers to dynamic name-based lookup with aliases.

## Static Typed Provider Selection (Recommended)

Use static typed providers for compile-time safety and IDE autocomplete:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // Static typed provider selection
  final openaiProvider = ChatProvider.openai;
  final anthropicProvider = ChatProvider.anthropic;
  final googleProvider = ChatProvider.google;
  final mistralProvider = ChatProvider.mistral;
  final cohereProvider = ChatProvider.cohere;
  final ollamaProvider = ChatProvider.ollama;

  // Create agents from providers
  final agent = Agent.fromProvider(openaiProvider);
  final result = await agent.run('Hello, world!');
  print(result.output);
}
```

## Dynamic Provider Selection by Name

Use dynamic lookup for configuration-driven applications:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // Dynamic selection by name
  final provider = ChatProvider.forName('openai');
  final agent = Agent.fromProvider(provider);
  
  // Direct agent creation with provider:model string
  final directAgent = Agent('openai:gpt-4o');
  
  final result = await directAgent.run('Hello, world!');
  print(result.output);
}
```

## Provider Aliases

Dartantic supports intuitive aliases for popular providers:

```dart
// Using aliases for common providers
final claudeProvider = ChatProvider.forName('claude');    // → Anthropic
final geminiProvider = ChatProvider.forName('gemini');    // → Google
final llamaProvider = ChatProvider.forName('llama');      // → Ollama

// Create agents with aliases
final claudeAgent = Agent('claude:claude-3-5-sonnet');
final geminiAgent = Agent('gemini:gemini-2.0-flash');
```

## Environment-Based Selection

Select providers based on environment variables:

```dart
import 'dart:io';
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // Environment-based provider selection
  final providerName = Platform.environment['AI_PROVIDER'] ?? 'openai';
  final modelName = Platform.environment['AI_MODEL'] ?? 'gpt-4o-mini';
  
  final agent = Agent('$providerName:$modelName');
  final result = await agent.run('Hello, world!');
  print(result.output);
}
```

## Provider Capability Checking

Check what features a provider supports before using them:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  final provider = ChatProvider.forName('openai');
  
  // Check provider capabilities
  print('Provider: ${provider.displayName}');
  print('Capabilities: ${provider.caps}');
  
  // Check specific capabilities
  if (provider.caps.contains(ProviderCaps.embeddings)) {
    print('✅ Supports embeddings');
  } else {
    print('❌ No embedding support');
  }
  
  if (provider.caps.contains(ProviderCaps.vision)) {
    print('✅ Supports vision/images');
  } else {
    print('❌ No vision support');
  }
}
```

## Multi-Provider Applications

Build applications that work with multiple providers:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // List all available providers
  final providers = ChatProvider.all;
  
  for (final provider in providers) {
    print('Testing ${provider.displayName}...');
    
    try {
      final agent = Agent.fromProvider(provider);
      final result = await agent.run('Say hello');
      print('✅ ${provider.displayName}: ${result.output}');
    } catch (e) {
      print('❌ ${provider.displayName}: $e');
    }
  }
}
```

## Provider Comparison Example

Compare responses across multiple providers:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

Future<void> compareProviders(String prompt) async {
  final providers = ['openai', 'anthropic', 'google'];
  
  for (final providerName in providers) {
    try {
      final agent = Agent('$providerName');
      final result = await agent.run(prompt);
      
      print('--- ${providerName.toUpperCase()} ---');
      print(result.output);
      print('');
    } catch (e) {
      print('$providerName failed: $e');
    }
  }
}

void main() async {
  await compareProviders('Explain quantum computing in one sentence');
}
```

## Best Practices

### For Development
1. **Use static providers** for type safety during development
2. **Test with Ollama** for free local development
3. **Use aliases** for shorter, more readable code

### For Production
1. **Use environment variables** for provider configuration
2. **Implement fallback providers** for reliability
3. **Check capabilities** before using features
4. **Monitor provider performance** and costs

### For Multi-tenant Applications
1. **Allow per-user provider selection**
2. **Implement provider quotas and limits**
3. **Cache provider capabilities** for performance

## Error Handling

Handle provider selection errors gracefully:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  try {
    // This will throw if provider doesn't exist
    final provider = ChatProvider.forName('nonexistent');
  } catch (e) {
    print('Provider not found: $e');
    
    // Fallback to default provider
    final fallbackProvider = ChatProvider.openai;
    final agent = Agent.fromProvider(fallbackProvider);
  }
}
```
