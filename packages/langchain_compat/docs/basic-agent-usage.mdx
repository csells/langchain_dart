# Basic Agent Usage

Create agents that handle both chat and embeddings.

## Create an Agent

```dart
// By provider
final agent = Agent('openai');

// With model
final agent = Agent('anthropic:claude-3-5-sonnet-latest');

// With options
final agent = Agent(
  'google',
  temperature: 0.7,
  systemPrompt: 'Be concise.',
);
```

## Chat

```dart
// Simple
final result = await agent.send('Hi');
print(result.output);

// With history
final history = <ChatMessage>[];
final result = await agent.send('Hi', history: history);
history.addAll(result.messages);

// Stream
await for (final chunk in agent.sendStream('Hi')) {
  stdout.write(chunk.output);
}
```

## Embeddings

```dart
// Single text
final result = await agent.embedQuery('Hello');

// Multiple texts  
final results = await agent.embedDocuments(['Hi', 'Hey', 'Hello']);

// Compare
final similarity = EmbeddingsModel.cosineSimilarity(
  result.embeddings,
  results.embeddings[0],
);
```

## Model Formats

```dart
Agent('openai');                    // Default models
Agent('openai:gpt-4o');            // Specific chat model
Agent('openai?chat=gpt-4o&embeddings=text-embedding-3-large');
```
