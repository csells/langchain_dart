# Basic ChatAgent Usage

The ChatAgent class is the core of Dartantic AI.

## Creating ChatAgents

### Provider Names
```dart
final agent = ChatAgent('openai');    // OpenAI
final agent = ChatAgent('anthropic'); // Anthropic
final agent = ChatAgent('google');    // Google
final agent = ChatAgent('mistral');   // Mistral
final agent = ChatAgent('ollama');    // Ollama (local)
```

### Aliases
```dart
final agent = ChatAgent('claude');  // → anthropic
final agent = ChatAgent('gemini');  // → google
```

### Specific Models
```dart
// Format: provider:model
final agent = ChatAgent('openai:gpt-4o');
final agent = ChatAgent('anthropic:claude-3-5-sonnet-latest');
final agent = ChatAgent('google:gemini-2.0-flash');
```

## Configuration

```dart
final agent = ChatAgent(
  'openai',
  temperature: 0.7,        // 0-1, higher = more creative
  systemPrompt: 'You are a helpful assistant.',
  tools: [weatherTool],    // Optional tools
);
```

## Using ChatAgents

### Basic Chat
```dart
final result = await agent.run('Hello!');
print(result.output);
```

### Streaming
```dart
await for (final chunk in agent.runStream('Tell me a story')) {
  stdout.write(chunk.output);
}
```

### With History
```dart
final history = <ChatMessage>[];

// First message
var result = await agent.run('My name is Alice', history: history);
history.addAll(result.messages);

// Agent remembers
result = await agent.run("What's my name?", history: history);
print(result.output); // "Your name is Alice"
```

## Advanced Usage

### Provider Discovery
```dart
// List all providers
for (final provider in ChatProvider.all) {
  print('${provider.name}: ${provider.defaultModelName}');
}

// Find by capability
final visionProviders = ChatProvider.allWith({ProviderCaps.vision});
```

### Direct Provider Access
```dart
final provider = ChatProvider.forName('openai');
final agent = ChatAgent.forProvider(provider, modelName: 'gpt-4o');
```
