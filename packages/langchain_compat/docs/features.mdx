# Features

Dartantic provides a unified interface to 15+ LLM providers with powerful features.

## Core Features

### 🤖 Multi-Provider Support
Access 15+ providers through one API.
```dart
Agent('openai')     // OpenAI GPT models
Agent('anthropic')  // Claude models
Agent('google')     // Gemini models
```
[Learn more →](/providers)

### 🔄 Streaming Responses
Real-time output as it's generated.
```dart
await for (final chunk in agent.sendStream('Tell a story')) {
  stdout.write(chunk.output);
}
```
[Learn more →](/streaming-output)

### 💬 Multi-Turn Conversations
Maintain context across messages.
```dart
final history = <ChatMessage>[];
var result = await agent.send('Hi!', history: history);
history.addAll(result.messages);
```
[Learn more →](/multi-turn-chat)

### 📊 Typed Output
Get structured data from LLMs.
```dart
final weather = await agent.sendFor<Weather>(
  'Current weather?',
  outputSchema: weatherSchema,
  outputFromJson: Weather.fromJson,
);
```
[Learn more →](/typed-output)

### 🛠️ Tool Calling
Extend agents with custom functions.
```dart
final agent = Agent('openai', tools: [weatherTool]);
await agent.send('Weather in NYC?');
```
[Learn more →](/tool-calling)

### 🖼️ Multi-Media Input
Process images and files alongside text.
```dart
await agent.send(
  'What\'s in this image?',
  attachments: [await DataPart.fromFile(imageFile)],
);
```
[Learn more →](/multi-media-input)

### 🧮 Embeddings
Generate and compare vector embeddings.
```dart
final embedding = await agent.embedQuery('Hello world');
final similarity = EmbeddingsModel.cosineSimilarity(e1, e2);
```
[Learn more →](/embeddings)

### 🔌 MCP Support
Integrate with Model Context Protocol servers.
```dart
final mcpClient = McpClient('server-name', ['npx', 'server']);
await mcpClient.start();
```
[Learn more →](/mcp-support)

### 🔄 Automatic Retry
Built-in retry logic for rate limits and failures.
```dart
// Automatic - no configuration needed
final result = await agent.send('Hello!');
```
[Learn more →](/automatic-retry)

### 📈 Usage Tracking
Monitor token usage and costs.
```dart
print('Tokens: ${result.usage.totalTokens}');
```
[Learn more →](/usage-tracking)

### 🔍 Model Discovery
List available models and capabilities.
```dart
await for (final model in Provider.openai.listModels()) {
  print('${model.name}: ${model.displayName}');
}
```
[Learn more →](/providers)

### 🤖 Agentic Behavior
Multi-step reasoning and tool chaining.
```dart
// Agent chains tools automatically
final agent = Agent('openai', tools: [tool1, tool2]);
await agent.send('Complex task requiring multiple steps');
```
[Learn more →](/agentic-behavior)
