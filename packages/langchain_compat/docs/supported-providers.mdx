# Supported Providers

Dartantic AI supports **15 provider configurations** through 5 native implementations and 6 OpenAI-compatible endpoints.

## Quick Reference

### Chat Providers (11)
- **Native (5)**: OpenAI, Anthropic, Google, Mistral, Ollama
- **OpenAI-compatible (6)**: OpenRouter, Together AI, Cohere, Lambda, Google-OpenAI, Ollama-OpenAI

### Embeddings Providers (4)
OpenAI, Google, Mistral, Cohere

For complete details, see [Providers](/providers).

## Setup Examples

```dart
// Cloud providers
final openai = ChatAgent('openai:gpt-4o');
final claude = ChatAgent('claude'); // alias for anthropic
final gemini = ChatAgent('gemini'); // alias for google

// Local provider
final ollama = ChatAgent('ollama:llama3.2');

// Multi-model gateway
final openrouter = ChatAgent('openrouter:anthropic/claude-3.5-sonnet');
```

## Choosing a Provider

- **General purpose**: OpenAI or Anthropic
- **Fast & cheap**: Google Gemini Flash
- **Local/offline**: Ollama
- **Access to many models**: OpenRouter
- **European compliance**: Mistral

See [Provider Capabilities](/provider-capabilities) for feature comparison.
