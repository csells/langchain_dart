# Supported Providers

🚀 **Dartantic supports 11 major AI providers with access to over 750+ models** - the most comprehensive provider ecosystem in the Dart AI space!

Unlike other frameworks limited to 2-3 providers, Dartantic gives you the flexibility to choose the best model for your specific use case, from cloud giants to local inference.

## Quick Setup Examples

### Cloud Providers
```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // OpenAI (most popular)
  final openaiAgent = Agent('openai:gpt-4o');
  
  // Anthropic Claude (best reasoning)
  final claudeAgent = Agent('anthropic:claude-3-5-sonnet');
  
  // Google Gemini (best multimodal)
  final geminiAgent = Agent('google:gemini-2.0-flash');
  
  // OpenRouter (access to 300+ models)
  final openrouterAgent = Agent('openrouter:google/gemini-2.5-flash');
  
  final result = await openaiAgent.run('Hello, world!');
  print(result.output);
}
```

### Local Models with Ollama
```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // Native Ollama API
  final ollamaAgent = Agent('ollama:llama3.1');
  
  // OpenAI-compatible endpoint  
  final ollamaOpenAIAgent = Agent('ollama-openai:llama3.1');
  
  final result = await ollamaAgent.run('Explain quantum computing');
  print(result.output);
}
```

### Provider-Specific Features
```dart
// Mistral with embeddings
final mistralAgent = Agent('mistral:mistral-small-latest');
final embedding = await mistralAgent.createEmbedding('Hello world');

// Cohere with advanced embeddings
final cohereAgent = Agent('cohere:command-r-plus');
final cohereEmbedding = await cohereAgent.createEmbedding('Hello world');

// Together AI with Llama models
final togetherAgent = Agent('together:meta-llama/Llama-3.2-3B-Instruct-Turbo');
```

## Provider Architectures

### OpenAI-Compatible Providers
These providers use the OpenAI API format, making them easy to swap:
- **OpenAI** - The original
- **OpenRouter** - Aggregator with 300+ models
- **Together AI** - Open source models
- **Lambda** - High-performance inference
- **Ollama (OpenAI-compat)** - Local OpenAI compatibility
- **Google (OpenAI-compat)** - Gemini via OpenAI format

### Native API Providers  
These use provider-specific APIs for optimal performance:
- **Google Gemini** - Native Gemini API with advanced features
- **Anthropic** - Native Claude API with conversation optimization
- **Mistral** - Native Mistral API with embeddings
- **Cohere** - Native Cohere API with advanced embeddings
- **Ollama** - Native Ollama API for local models

## Choosing the Right Provider

### For Production Apps
- **OpenAI**: Most reliable, extensive ecosystem
- **Anthropic**: Best reasoning and safety
- **Google Gemini**: Best multimodal capabilities

### For Cost Optimization
- **OpenRouter**: Compare 300+ models and pricing
- **Together AI**: Open source models at low cost
- **Ollama**: Free local inference

### For Specialized Use Cases
- **Mistral**: European provider with strong embeddings
- **Cohere**: Enterprise-focused with advanced embeddings
- **Lambda**: High-performance inference for demanding workloads

### For Development & Testing
- **Ollama**: Free local development
- **OpenRouter**: Easy model comparison
- **Google Gemini**: Generous free tier
