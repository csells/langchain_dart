# Streaming Output

Get responses as they're generated.

```dart
final agent = Agent('openai');

await for (final chunk in agent.sendStream('Tell a story')) {
  stdout.write(chunk.output); // Prints as it streams
}
```

## With History

```dart
await for (final chunk in agent.sendStream('Continue', history: history)) {
  stdout.write(chunk.output);
}
```

See [example/chat.dart](https://github.com/davidmigloz/langchain_dart/blob/main/packages/langchain_compat/example/chat.dart).
