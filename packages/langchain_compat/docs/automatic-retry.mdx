# Automatic Retry Logic

Dartantic includes sophisticated automatic retry logic with exponential backoff to handle rate limits, temporary network issues, and API timeouts gracefully, ensuring your applications remain robust and reliable.

## Built-in Retry Features

The retry system is automatically enabled for all OpenAI-compatible providers with zero configuration required:

```dart
import 'package:dartantic_ai/dartantic_ai.dart';

void main() async {
  // Retry is automatic - no configuration needed
  final agent = Agent('openai:gpt-4o-mini');
  
  // This will automatically retry on 429 (rate limit) errors
  final result = await agent.run('Hello, world!');
  print(result.output);
}
```

## Retry Capabilities

### Automatic Rate Limit Handling
- **429 Error Detection**: Automatically detects rate limit responses
- **Retry-After Header Support**: Respects server-provided retry delays
- **Exponential Backoff**: Implements smart delay escalation
- **Configurable Attempts**: Default 3 retry attempts with customization options

### Supported Error Types
- **Rate Limits (429)**: Primary focus with intelligent retry timing
- **Server Errors (5xx)**: Temporary server issues
- **Network Timeouts**: Connection and read timeout handling
- **Connection Failures**: Temporary network connectivity issues

## How It Works

### Retry Process
1. **Initial Request**: Make the original API request
2. **Error Detection**: Identify retryable errors (429, 5xx, timeouts)
3. **Delay Calculation**: Calculate appropriate delay (exponential backoff + jitter)
4. **Retry-After Respect**: Honor server-provided retry timing when available
5. **Request Recreation**: Create fresh request with updated headers/auth
6. **Retry Execution**: Execute retry with proper logging

### Timing Strategy
```
Attempt 1: Immediate
Attempt 2: 1 second + jitter
Attempt 3: 2 seconds + jitter  
Attempt 4: 4 seconds + jitter
Max delay: 60 seconds
```

## Best Practices for Retry-Aware Applications

### Design for Resilience
```dart
class ResilientAgent {
  final Agent _agent;
  final int _maxApplicationRetries;
  
  ResilientAgent(String model, {int maxApplicationRetries = 2})
      : _agent = Agent(model),
        _maxApplicationRetries = maxApplicationRetries;
  
  Future<String> generateWithFallback(String prompt) async {
    for (int attempt = 0; attempt < _maxApplicationRetries; attempt++) {
      try {
        final result = await _agent.run(prompt);
        return result.output;
      } catch (e) {
        if (attempt == _maxApplicationRetries - 1) rethrow;
        
        // Application-level backoff (on top of automatic HTTP retries)
        final delay = Duration(seconds: (attempt + 1) * 30);
        print('Attempt ${attempt + 1} failed, retrying in ${delay.inSeconds}s');
        await Future.delayed(delay);
      }
    }
    
    throw Exception('All retry attempts exhausted');
  }
}

void main() async {
  final resilientAgent = ResilientAgent('openai:gpt-4o-mini');
  
  try {
    final result = await resilientAgent.generateWithFallback(
      'Write a detailed market analysis'
    );
    print('Generated: ${result.substring(0, 100)}...');
  } catch (e) {
    print('Failed with maximum resilience: $e');
  }
}
```

### Monitor Retry Patterns
```dart
class RetryMonitor {
  static final Map<String, int> _retryCount = {};
  static final Map<String, DateTime> _lastRetry = {};
  
  static void recordRetry(String provider) {
    _retryCount[provider] = (_retryCount[provider] ?? 0) + 1;
    _lastRetry[provider] = DateTime.now();
  }
  
  static void printStats() {
    print('=== Retry Statistics ===');
    for (final entry in _retryCount.entries) {
      final lastRetry = _lastRetry[entry.key];
      print('${entry.key}: ${entry.value} retries, last: $lastRetry');
    }
  }
}

    }
  });
  
  // Your application code here
  // ...
  
  RetryMonitor.printStats();
}
```

### Optimize for Retry Success
```dart
class RetryOptimizedAgent {
  final Agent _agent;
  
  RetryOptimizedAgent(String model) : _agent = Agent(model);
  
  Future<String> optimizedGenerate(String prompt) async {
    // Shorter prompts are less likely to hit rate limits
    final optimizedPrompt = prompt.length > 1000 
        ? '${prompt.substring(0, 1000)}...'
        : prompt;
    
    return (await _agent.run(optimizedPrompt)).output;
  }
  
  Future<List<String>> batchGenerate(List<String> prompts) async {
    final results = <String>[];
    
    // Process in smaller batches to avoid overwhelming the API
    const batchSize = 3;
    for (int i = 0; i < prompts.length; i += batchSize) {
      final batch = prompts.skip(i).take(batchSize);
      
      for (final prompt in batch) {
        final result = await optimizedGenerate(prompt);
        results.add(result);
        
        // Small delay between requests to be API-friendly
        await Future.delayed(Duration(milliseconds: 500));
      }
    }
    
    return results;
  }
}
```

## Retry Configuration by Use Case

### High-Volume Applications
```dart
// For applications making many requests
void main() async {
  // Use faster models to reduce rate limit pressure
  final agent = Agent('openai:gpt-4o-mini'); // Faster than gpt-4o
  
  // Implement application-level request queuing
  final requestQueue = <Future<String>>[];
  
  for (int i = 0; i < 100; i++) {
    requestQueue.add(
      Future.delayed(
        Duration(milliseconds: i * 100), // Spread requests over time
        () => agent.run('Request $i').then((r) => r.output),
      ),
    );
  }
  
  final results = await Future.wait(requestQueue);
  print('Completed ${results.length} requests');
}
```

### Critical Applications
```dart
// For applications where failures are not acceptable
Future<String> criticalGenerate(String prompt) async {
  final providers = [
    'openai:gpt-4o-mini',
    'anthropic:claude-3-5-haiku',
    'google:gemini-2.0-flash',
  ];
  
  for (final provider in providers) {
    try {
      final agent = Agent(provider);
      final result = await agent.run(prompt);
      return result.output;
    } catch (e) {
      print('$provider failed, trying next: $e');
      continue;
    }
  }
  
  throw Exception('All providers failed');
}
```
