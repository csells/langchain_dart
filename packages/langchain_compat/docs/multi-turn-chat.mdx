# Multi-turn Chat

Build context-aware conversations that remember previous messages.

## Basic Conversation

```dart
final agent = Agent('openai');
final history = <ChatMessage>[];

// First turn
var result = await agent.run(
  'My name is Alice and I work as a software engineer.',
  history: history,
);
history.addAll(result.messages);

// Second turn - agent remembers context
result = await agent.run(
  'What do you remember about me?',
  history: history,
);
history.addAll(result.messages);
print(result.output); // "Your name is Alice and you work as a software engineer."
```

## Message Management

Each `run()` returns all new messages:

```dart
final result = await agent.run('Hello!', history: history);
history.addAll(result.messages);

// result.messages contains:
// 1. User message: "Hello!"
// 2. Model's response message
```

## Multi-Provider Conversations

Share conversations between different providers:

```dart
final history = <ChatMessage>[];

// Start with fast Gemini
final gemini = Agent('google');
var result = await gemini.run('Hi, I'm learning Dart', history: history);
history.addAll(result.messages);

// Switch to Claude for complex reasoning
final claude = Agent('anthropic');
result = await claude.run('What should I build first?', history: history);
history.addAll(result.messages);

// Use OpenAI for code generation
final openai = Agent('openai');
result = await openai.run('Show me the code', history: history);
history.addAll(result.messages);
```

## Provider Selection Strategy

```dart
// Cost-optimized workflow
final simple = Agent('google:gemini-2.0-flash');    // Fast & cheap
```

## With Tools

Tools work seamlessly in conversations:

```dart
final agent = Agent('openai', tools: [weatherTool, calculatorTool]);
final history = <ChatMessage>[];

// Tools are available throughout the conversation
result = await agent.run('Check weather in NYC', history: history);
history.addAll(result.messages);

result = await agent.run('Is that warm in Celsius?', history: history);
history.addAll(result.messages);
```
