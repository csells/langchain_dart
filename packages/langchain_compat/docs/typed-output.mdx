# Typed Output

Get structured data from AI models instead of plain text.

## Basic JSON

```dart
final result = await agent.run(
  'Extract: The Eiffel Tower is in Paris, France',
  outputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
    'country': JsonSchema.string(),
  }),
);

print(result.output); // {"city":"Paris","country":"France"}

// Parse manually
final data = jsonDecode(result.output);
print('City: ${data['city']}');
```

## Map Output

Get a parsed Map automatically:

```dart
final result = await agent.runFor<Map<String, dynamic>>(
  'List 3 programming languages',
  outputSchema: JsonSchema.object({
    'languages': JsonSchema.array(
      items: JsonSchema.object({
        'name': JsonSchema.string(),
        'year': JsonSchema.integer(),
      }),
    ),
  }),
);

// Direct Map access
final languages = result.output['languages'] as List;
```

## Custom Types

```dart
class Weather {
  final String location;
  final double temperature;
  
  Weather.fromJson(Map<String, dynamic> json)
    : location = json['location'],
      temperature = json['temperature'];
      
  static final schema = JsonSchema.object({
    'location': JsonSchema.string(),
    'temperature': JsonSchema.number(),
  });
}

final result = await agent.runFor<Weather>(
  'Weather in Seattle?',
  outputSchema: Weather.schema,
  outputFromJson: Weather.fromJson,
);

print('${result.output.location}: ${result.output.temperature}°');
```

## With Tools

The `return_result` tool enables typed output alongside other tools:

```dart
final returnResultTool = Tool.returnResult(JsonSchema.object({
  'summary': JsonSchema.string(),
  'wordCount': JsonSchema.integer(),
}));

final agent = Agent('openai', tools: [
  calculatorTool,
  returnResultTool,
]);

final result = await agent.runFor<Map<String, dynamic>>(
  'Calculate 15% tip on $45 and summarize',
);
```

## Provider Support

- **Full support**: OpenAI, Anthropic
- **JSON only**: Google, Ollama, most OpenAI-compatible
- **Limited**: Mistral (no typed output)

See [Providers](/providers) for capability details.
