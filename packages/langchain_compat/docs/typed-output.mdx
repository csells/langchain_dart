# Typed Output

Get structured data from AI models.

## Basic JSON

```dart
final result = await agent.send(
  'Extract city: Paris is beautiful',
  outputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
  }),
);

print(result.output); // {"city":"Paris"}
```

## Parsed Map

```dart
final result = await agent.sendFor<Map<String, dynamic>>(
  'List 3 colors',
  outputSchema: JsonSchema.object({
    'colors': JsonSchema.array(
      items: JsonSchema.string(),
    ),
  }),
);

print(result.output['colors']); // ['red', 'blue', 'green']
```

## Custom Types

```dart
class Weather {
  final String city;
  final int temp;
  
  Weather.fromJson(Map<String, dynamic> json)
    : city = json['city'],
      temp = json['temp'];
}

final result = await agent.sendFor<Weather>(
  'Weather in NYC?',
  outputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
    'temp': JsonSchema.integer(),
  }),
  outputFromJson: Weather.fromJson,
);

print('${result.output.city}: ${result.output.temp}°');
```

## With Tools

```dart
// Typed output + tool calls in one request
final result = await agent.sendFor<Recipe>(
  'Get grandma\'s recipe for cookies',
  outputSchema: recipeSchema,
  outputFromJson: Recipe.fromJson,
  tools: [recipeLookupTool],
);

print('${result.output.name}: ${result.output.ingredients}');
```

## Streaming

```dart
// Stream structured JSON
await for (final chunk in agent.sendStream(
  'List 3 facts',
  outputSchema: factsSchema,
)) {
  stdout.write(chunk.output); // Streams JSON chunks
}
```
