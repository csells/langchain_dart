# Chat Class

The Chat class provides a simplified interface for managing multi-turn conversations with Agents.

## Overview

The Chat class wraps a Agent and automatically maintains conversation history, making it easy to build interactive chat sessions without manually managing message lists.

## Creating a Chat Instance

```dart
import 'package:langchain_compat/langchain_compat.dart';

// Create a Agent first
final agent = Agent('openai:gpt-4o-mini');

// Create a Chat instance
final chat = Chat(agent);

// Or with initial history
final chatWithHistory = Chat(
  agent,
  history: [
    ChatMessage.user('Hello!'),
    ChatMessage.model('Hi there! How can I help you today?'),
  ],
);
```

## Basic Usage

### Simple Conversation

```dart
// Send a message and get a response
final result = await chat.send('What is the capital of France?');
print(result.output); // "The capital of France is Paris."

// Follow-up question - history is maintained automatically
final followUp = await chat.send('What is its population?');
print(followUp.output); // "Paris has a population of approximately 2.2 million..."
```

### With Attachments

```dart
// Send an image with a question
final imageData = await File('photo.jpg').readAsBytes();
final attachment = DataPart(
  data: imageData,
  mimeType: 'image/jpeg',
);

final result = await chat.send(
  'What do you see in this image?',
  attachments: [attachment],
);
```

### With Output Schema

```dart
// Request structured output
final schema = JsonSchema.create({
  'type': 'object',
  'properties': {
    'name': {'type': 'string'},
    'population': {'type': 'number'},
  },
  'required': ['name', 'population'],
});

final result = await chat.send(
  'Give me info about Paris',
  outputSchema: schema,
);
```

## Typed Output

Use `sendFor<T>()` to get strongly-typed responses:

```dart
// Define a type
class CityInfo {
  CityInfo({required this.name, required this.population});
  
  factory CityInfo.fromJson(Map<String, dynamic> json) => CityInfo(
    name: json['name'] as String,
    population: json['population'] as int,
  );
  
  final String name;
  final int population;
}

// Get typed response
final result = await chat.sendFor<CityInfo>(
  'Give me info about Paris',
  outputSchema: JsonSchema.create({
    'type': 'object',
    'properties': {
      'name': {'type': 'string'},
      'population': {'type': 'integer'},
    },
    'required': ['name', 'population'],
  }),
  outputFromJson: CityInfo.fromJson,
);

print(result.output.name);       // "Paris"
print(result.output.population); // 2200000
```

## Streaming Responses

```dart
// Stream responses for real-time output
await for (final chunk in chat.sendStream('Tell me a story')) {
  stdout.write(chunk.output); // Print each chunk as it arrives
}

// History is still maintained after streaming
print(chat.history.length); // Will show all messages including the story
```

## Using with Tools

```dart
import 'package:langchain_compat/example.dart';

// Create agent with tools
final agent = Agent(
  'openai:gpt-4o-mini',
  tools: [weatherTool, temperatureConverterTool],
);

final chat = Chat(agent);

// The agent will use tools automatically
final result = await chat.send(
  'What is the weather in Paris and convert it to Fahrenheit?'
);
// The agent will call weatherTool and temperatureConverterTool automatically
```

## Accessing Conversation History

```dart
// The history property contains all messages
for (final message in chat.history) {
  print('${message.role}: ${message.text}');
}

// Check history length
print('Messages in conversation: ${chat.history.length}');

// History includes both user and model messages
final userMessages = chat.history.where((m) => m.role == ChatMessageRole.user);
final modelMessages = chat.history.where((m) => m.role == ChatMessageRole.model);
```

## Complete Example

```dart
import 'package:langchain_compat/langchain_compat.dart';
import 'package:langchain_compat/example.dart';

void main() async {
  // Create an agent with tools
  final agent = Agent(
    'anthropic:claude-3-5-sonnet-latest',
    tools: [weatherTool],
  );
  
  // Create a chat session
  final chat = Chat(agent);
  
  // Have a conversation
  print('User: What\'s the weather like in London?');
  final weather = await chat.send('What\'s the weather like in London?');
  print('Assistant: ${weather.output}');
  
  print('\nUser: Is that warm for this time of year?');
  final analysis = await chat.send('Is that warm for this time of year?');
  print('Assistant: ${analysis.output}');
  
  // Get typed output
  print('\nUser: Give me the time and temperature in Paris');
  final typedResult = await chat.sendFor<TimeAndTemperature>(
    'What is the current time and temperature in Paris?',
    outputSchema: TimeAndTemperature.schema,
    outputFromJson: TimeAndTemperature.fromJson,
  );
  
  print('Time: ${typedResult.output.time}');
  print('Temperature: ${typedResult.output.temperature}°C');
  
  // Show conversation history
  print('\nConversation history:');
  for (final message in chat.history) {
    final role = message.role == ChatMessageRole.user ? 'User' : 'Assistant';
    print('$role: ${message.text}');
  }
}
```

## Key Benefits

1. **Automatic History Management**: No need to manually track messages
2. **Simplified API**: Clean methods for common operations
3. **Type Safety**: Strong typing support with `sendFor<T>()`
4. **Streaming Support**: Built-in streaming with history preservation
5. **Tool Integration**: Works seamlessly with Agent tools
