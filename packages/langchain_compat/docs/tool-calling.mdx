# Tool Calling

Extend AI chat agents with custom functions to access real-time data, perform calculations, and interact with external systems.

## Basic Tool

```dart
final weatherTool = Tool(
  name: 'get_weather',
  description: 'Get weather for a city',
  inputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
  }),
  onCall: (args) async => {
    'city': args['city'],
    'temperature': 72,
    'conditions': 'Sunny',
  },
);

final agent = ChatAgent('openai', tools: [weatherTool]);
final result = await agent.run("What's the weather in Seattle?");
```

## Tool Without Parameters

```dart
final timeTool = Tool(
  name: 'get_time',
  description: 'Get current UTC time',
  onCall: (_) async => {
    'time': DateTime.now().toUtc().toIso8601String(),
  },
);
```

## Multiple Tools

ChatAgents automatically select the right tools:

```dart
final agent = ChatAgent('anthropic', tools: [
  weatherTool,
  timeTool,
  calculatorTool,
]);

// Uses multiple tools as needed
final result = await agent.run(
  "What's the weather in Paris and what time is it?"
);
```

## Parallel Tool Calls

Some providers support calling multiple tools at once:

```dart
// OpenAI and Anthropic support parallel calls
final result = await agent.run(
  "Check weather in NYC, London, and Tokyo"
);
```

## Error Handling

```dart
final safeTool = Tool(
  name: 'divide',
  description: 'Divide two numbers',
  inputSchema: JsonSchema.object({
    'a': JsonSchema.number(),
    'b': JsonSchema.number(),
  }),
  onCall: (args) async {
    final b = args['b'] as num;
    if (b == 0) {
      throw ToolException('Cannot divide by zero');
    }
    return {'result': (args['a'] as num) / b};
  },
);
```

## MCP Integration

Connect to Model Context Protocol servers:

```dart
final mcpClient = McpClient(
  'filesystem',
  arguments: ['/path/to/server'],
);
await mcpClient.connect();

final agent = ChatAgent('claude', tools: mcpClient.tools);
```

See [example/mcp_client.dart](https://github.com/davidmigloz/langchain_dart/blob/main/packages/langchain_compat/example/mcp_client.dart) for details.

## Provider Support

- **Full support**: OpenAI, Anthropic, Google, Ollama
- **Limited**: Cohere (single tool), OpenRouter (varies by model)
- **None**: Mistral, Together AI

Check with `provider.caps.contains(ProviderCaps.multiToolCalls)`.
