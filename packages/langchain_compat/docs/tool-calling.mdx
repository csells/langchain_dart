# Tool Calling

Extend agents with custom functions.

## Basic Tool

```dart
final weatherTool = Tool(
  name: 'get_weather',
  description: 'Get weather for a city',
  inputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
  }),
  onCall: (args) async => {
    'temperature': 72,
    'conditions': 'Sunny',
    'city': args['city'],
  },
);

final agent = Agent('openai', tools: [weatherTool]);
final result = await agent.send("Weather in Seattle?");
print(result.output); // "The weather in Seattle is sunny with a temperature of 72°F"
```

## Multiple Tools

```dart
final agent = Agent('anthropic', tools: [
  weatherTool,
  temperatureConverterTool,
  currentDateTimeTool,
]);

// Single request, multiple tool calls
final result = await agent.send(
  "What's the weather in Paris? Convert to Fahrenheit and tell me the time."
);
// Agent calls all three tools automatically
```

## Streaming Tool Calls

```dart
await for (final chunk in agent.sendStream(
  "Weather in NYC and Boston?",
)) {
  stdout.write(chunk.output); // Streams response as tools execute
}
```

## Error Handling

```dart
final tool = Tool(
  name: 'divide',
  description: 'Divide numbers',
  inputSchema: JsonSchema.object({
    'a': JsonSchema.number(),
    'b': JsonSchema.number(),
  }),
  onCall: (args) async {
    if (args['b'] == 0) throw 'Cannot divide by zero';
    return {'result': args['a'] / args['b']};
  },
);
```

```dart

provider.caps.contains(ProviderCaps.multiToolCalls)
```
