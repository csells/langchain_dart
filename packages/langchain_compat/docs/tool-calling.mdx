# Tool Calling

Extend agents with custom functions.

## Basic Tool

```dart
final weatherTool = Tool(
  name: 'get_weather',
  description: 'Get weather for a city',
  inputSchema: JsonSchema.object({
    'city': JsonSchema.string(),
  }),
  onCall: (args) async => {
    'temperature': 72,
    'conditions': 'Sunny',
  },
);

final agent = Agent('openai', tools: [weatherTool]);
final result = await agent.send("Weather in Seattle?");
```

## Multiple Tools

```dart
final agent = Agent('anthropic', tools: [
  weatherTool,
  timeTool,
  calculatorTool,
]);

final result = await agent.send(
  "Weather in Paris and current time?"
);
```

## Error Handling

```dart
final tool = Tool(
  name: 'divide',
  description: 'Divide numbers',
  inputSchema: JsonSchema.object({
    'a': JsonSchema.number(),
    'b': JsonSchema.number(),
  }),
  onCall: (args) async {
    if (args['b'] == 0) throw 'Cannot divide by zero';
    return {'result': args['a'] / args['b']};
  },
);
```

## Provider Support

- **Full**: OpenAI, Anthropic, Google, Ollama
- **Limited**: Cohere, OpenRouter
- **None**: Mistral, Together AI

Check: `provider.caps.contains(ProviderCaps.multiToolCalls)`
