# Dartantic

Hello and welcome to Dartantic!

The [dartantic_ai package](https://pub.dev/packages/dartantic_ai) is an agent framework inspired by pydantic-ai, designed to make building Dart and Flutter apps with AI easier and more fun!

## What is Dartantic AI?

One API, 15 provider configurations:
- **5 native implementations**: OpenAI, Google (Gemini), Anthropic (Claude), Mistral, Ollama
- **6 OpenAI-compatible**: OpenRouter, Together AI, Cohere, Lambda, Google-OpenAI, Ollama-OpenAI
- **4 embeddings providers**: OpenAI, Google, Mistral, Cohere

Switch providers with one line of code.

## Why Dartantic?

Dartantic was born from frustration - why use different APIs for different AI models in Dart? 

Consider [CalPal](https://github.com/csells/calpal), a Flutter app that manages calendars with natural language:

<Image src="/images/calpal.png" alt="CalPal screenshot" caption="CalPal in action" />

In ~300 lines, CalPal understands "What's on my calendar today?" by figuring out the local time and querying Google Calendar through tools. That's the power of an agentic framework.

## Quick Examples

### Basic Chat
```dart
import 'package:dartantic_ai/dartantic_ai.dart';

final agent = ChatAgent('claude'); // or 'openai', 'gemini', etc.
final result = await agent.run('Hello!');
print(result.output);
```

### Streaming
```dart
await for (final chunk in agent.runStream('Tell me a story')) {
  stdout.write(chunk.output);
}
```

### Tools
```dart
final weatherTool = Tool(
  name: 'get_weather',
  description: 'Get weather for a location',
  inputSchema: JsonSchema.object({
    'location': JsonSchema.string(),
  }),
  onCall: (args) async => {'temp': 72, 'condition': 'sunny'},
);

final agent = ChatAgent('openai', tools: [weatherTool]);
final result = await agent.run("Weather in Seattle?");
```

### Multi-Provider Conversations
```dart
final history = <ChatMessage>[];

// Start with fast Gemini
final gemini = ChatAgent('google');
final result1 = await gemini.run('Hi, I'm Alice', history: history);
history.addAll(result1.messages);

// Switch to Claude for reasoning
final claude = ChatAgent('anthropic');
final result2 = await claude.run('What's my name?', history: history);
print(result2.output); // "Your name is Alice"
```

## Key Features

- **Unified API**: Same code for all providers
- **ChatAgent-Based**: Automatic tool execution and state management
- **Multi-Provider**: Share conversations between providers
- **Streaming**: Real-time responses
- **Tools**: Extend with custom functions
- **Typed Output**: Get structured data back
- **MCP Support**: Connect to external tools
